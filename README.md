# Petrol Price Analysis: Data Analysis & BI Reporting

## Overview

This project leverages **IBM Cognos Analytics** to transform raw data into
actionable business intelligence. By utilizing advanced reporting and
interactive dashboards, the project identifies key trends, optimizes data
exploration, and provides a clear narrative for stakeholders.

---

## Objectives

* **Data Modeling:** Create a robust data module to ensure accurate reporting.
* **Interactive Visualization:** Design intuitive dashboards for real-time data
exploration.
* **Business Insights:** Extract high-impact findings to support data-driven
decision-making.
* **Technical Mastery:** Demonstrate proficiency in IBM Cognos tools, including
calculations, filters, and conditional formatting.

## Key Insights

* **Trend 1:** Loss values tend to vary across regions/categories, and visualizations like column and line charts help identify periods or segments where losses increase or decrease significantly.
* **Anomaly 2:** Certain categories or regions may show unexpectedly high or low loss values compared to others, indicating potential operational issues or irregular events that require further investigation.
* **Prediction 3:** Based on observed patterns, if a specific category or region continues to show increasing loss trends, it is likely to contribute more heavily to future overall losses unless corrective measures are taken.
